# 🚀 MAKAUT PPS Practice Bot

A Telegram bot designed to help 2nd semester MAKAUT students learn **C programming (PPS)** through a structured, day-wise practice system.

Unlike traditional approaches that rely on memorization, this bot focuses on **building problem-solving skills** and **confidence in writing code independently**.

---

## 🎯 Problem Statement

Many PPS students:
- Struggle to understand programming fundamentals
- Rely on memorizing code before exams
- Lack consistent daily practice

This bot addresses these issues with a **guided learning system** that promotes:
- Active thinking
- Step-by-step problem solving
- Daily consistency

---

## 🧠 Features

### 🚀 Daily Practice System
- 7-day structured learning path
- One problem per day (beginner → exam level)
- Designed for 20–30 minutes daily engagement

### 🪜 Guided Learning Flow
Each problem includes:
- Problem statement
- Hint 1 → Hint 2 → Final code
- Reflection questions to reinforce concepts

### 📊 Progress Tracking
- Tracks user progress across days
- Resume from last completed day

### ⏳ Time-Gated Learning
- Next day unlocks after **18 hours**
- Encourages consistency instead of binge learning

### 🔔 Reminder System *(planned / in progress)*
- Notification when next day unlocks
- Reminder if user is inactive

### 🎯 Exam Preparation *(upcoming)*
- Important programs
- Viva questions
- Revision notes

---

## 🏗️ Tech Stack

- **Language:** Python
- **Framework:** python-telegram-bot (v20+)
- **Storage:** JSON (for user data & content)
- **Deployment:** Render (planned)
